# 🚀 快速开始 - Tikkuu Focus

## ⚡ 5 分钟快速测试

### 1️⃣ 运行应用（1 分钟）
```bash
# 连接 iPhone 到 Mac
# 在 Xcode 中选择你的设备
# 按 Cmd+R
```

### 2️⃣ 授予权限（30 秒）
- 点击 "允许在使用应用时"
- 等待位置状态变为 "Location ready" ✅

### 3️⃣ 开始旅程（30 秒）
- 选择 "Cycling" 🚴
- 选择 "5 min" ⏱️
- 点击 "Start Journey" 🚀

### 4️⃣ 观察旅程（3 分钟）
- 看地图上的蓝色路线 🗺️
- 看虚拟角色移动 🔵
- 看计时器倒计时 ⏰

### 5️⃣ 完成测试（30 秒）
- 等待或点击 X 取消
- 返回设置界面
- ✅ 测试完成！

## 🐛 常见问题

### Q: 位置一直显示 "Getting location..."
**A:** 到窗边或室外，GPS 信号更好

### Q: 无法点击 "Start Journey"
**A:** 等待位置状态变为 "Location ready"

### Q: 旅程开始后报错
**A:** 检查网络连接，重试会生成新路线

### Q: 没有发现 POI
**A:** 正常，不是所有地方都有地标
